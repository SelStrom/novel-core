## 📋 Инструкция: Как настроить Sample Project

### Вариант 1: Автоматическая настройка (рекомендуется)

1. **Открыть Unity Editor**
2. **Меню**: `NovelCore` → `Generate Sample Project`
3. **Подтвердить**: Если проект уже существует, выбрать "Overwrite"
4. **Дождаться**: Сообщения "Sample Project Created"

Генератор автоматически:
- Создаёт все необходимые ассеты (сцены, фоны, персонажей)
- Импортирует изображения как Sprites
- Добавляет их в Addressables с правильными адресами
- Настраивает Unity сцену с UI и навигацией
- ✨ **НОВОЕ**: Больше не требует ручной настройки Addressables!

### Вариант 2: Ручная настройка

#### Шаг 1: Импортировать Sprites
1. Выбрать все `.png` в `Art/Backgrounds/` и `Art/Characters/`
2. Inspector → **Texture Type**: `Sprite (2D and UI)`
3. **Apply**

#### Шаг 2: Настроить Addressables
1. Выбрать `bg_scene1.png` → Inspector → ✓ **Addressable**
2. **Address**: `Backgrounds/bg_intro`
3. Повторить для остальных:
   - `bg_scene2.png` → `Backgrounds/bg_choice`
   - `bg_scene3.png` → `Backgrounds/bg_pathA`
   - `bg_scene4.png` → `Backgrounds/bg_pathB`
   - `character_placeholder.png` → `Characters/character_test`

#### Шаг 3: Обновить SceneData
1. Открыть `Scene01_Introduction.asset`
2. **Background Image**:
   - Нажать кружок справа
   - Выбрать `Backgrounds/bg_intro`
3. **Apply**
4. Повторить для:
   - `Scene02_ChoicePoint` → `Backgrounds/bg_choice`
   - `Scene03a_PathA` → `Backgrounds/bg_pathA`
   - `Scene03b_PathB` → `Backgrounds/bg_pathB`

## 🧪 Как протестировать

### 1. Проверка фонов и диалогов
```bash
1. Открыть SampleScene.unity
2. Нажать Play ▶️
3. Ожидаемый результат:
   ✓ Виден фоновый спрайт (синий)
   ✓ Диалоговое окно с текстом
   ✓ Можно кликать для продолжения диалога
```

### 2. Проверка навигации
```bash
1. Запустить игру (Play)
2. Пройти первую сцену (Scene01) → перейти к Scene02
3. Проверить:
   ✓ Кнопка "◀ Назад" становится БЕЛОЙ (активной)
   ✓ Кнопка "Вперёд →" серая (нет истории вперед)
4. Нажать "◀ Назад"
5. Проверить:
   ✓ Вернулись к Scene01
   ✓ Кнопка "Назад" снова серая
   ✓ Кнопка "Вперёд →" теперь БЕЛАЯ (можно вернуться к Scene02)
```

### Почему кнопки "не работают" на старте?
**Это ПРАВИЛЬНОЕ поведение!**

История навигации пустая при запуске:
```
Запуск → Scene01 [вы здесь]
         ↑ Нет предыдущих сцен → кнопка "Назад" неактивна

После перехода:
Scene01 ← Scene02 [вы здесь]
  ↑ Можно вернуться → кнопка "Назад" активна!
```
